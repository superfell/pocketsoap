PocketSOAP Versions

PocketSOAP is available in two major versions, a Win32 build that runs on the vast majority of Win32 platforms, and a pocketPC build that runs on pocketPC devices [aka WinCE].

Win 32

Platform Notes
Windows 2000 / XP / 2003
NT 4.0 Hasn't been tested on anything earlier than Service Pack 6
Win ME
Win 98 [all versions] SSL support requires additional installs from Microsoft, see this Knowledge Base article.
Win 95 OSR2 Requires DCOM 95 1.2 or later. SSL support requires additional installs from Microsoft, see this Knowledge Base article.
Win 95 pre OSR2 Will not run. This is because pocketSOAP uses a locale aware API for type conversions that was introduced with OSR2. Let me know if this is a serious issue, and i'll look into changing this.


pocketPC

Platform Notes
Pocket PC 2003 Runs on real PocketPC 2003 devices, I currently test with a Toshiba e355 running PocketPC 2003.
Pocket PC 2003 Emulator There's current no EVC4 based build, so its not possible to run it on the PPC2003 emulator.
Pocket PC 2002 Runs on real PocketPC 2002 devices, I currently test with an iPaq running PocketPC 2002, with a CF wireless card.
Pocket PC 2002 Emulator There are binaries but no installer for the PPC emulator. I don't normally test with the emulator as it is both flakey and behaves differently to a real PocketPC device.
Pocket PC An Installer that includes binaries for MIPS, ARM & SH3. is available.
Pocket PC Emulator There are binaries but no installer for the PPC emulator. I don't normally test with the emulator as it is both flakey and behaves differently to a real PocketPC device.
WIN CE 2.x There have been reports of pocketSOAP sucessfuly running on some WIN CE 2.x devices. It is unlikely that the source code will build correctly with the WIN CE 2.x development tools.
